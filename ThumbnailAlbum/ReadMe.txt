<< はじめに >>
-----------------------------------------------------------------------------------------
このたびは、HTML 形式アルバム作成ソフト「ThumbnailAlbum」をダウンロードしていただきまして
ありがとうございます。
ご利用になる前に、必ず使用権許諾の規定（license.txt）をお読みください。


<< インストール（起動） >>
-----------------------------------------------------------------------------------------
　1. ダウンロードしたファイルをダブルクリック（実行）すると、セットアップが開始されます。
　2. プログラムの指示に従って、セットアップしてください。


<< アンインストール（削除） >>
-----------------------------------------------------------------------------------------
　1. Windows の [スタート] ボタンをクリックし、[コントロール パネル]、[プログラム]、
　　 [プログラムと機能] の順にクリックします。
　　 ※お使いの Windows によっては、名称が異なる場合があります。ご注意ください。

　2. 登録されているプログラムの一覧が表示されますので、＜ThumbnailAlbum＞を選択し、
     [アンインストール] をクリックすると、セットアップが開始されます。

　3. プログラムの指示に従って、アンインストールしてください。


<< 設定情報の削除 >>
-----------------------------------------------------------------------------------------
　設定情報は、[AppData] フォルダー内のファイルを削除してください。

　例）C:\Users\{User Name}\AppData\Roaming\NJP\ThumbnailAlbum
　※お使いの Windows によっては、参照先が異なる場合があります。ご注意ください。


<< 使い方 >>
-----------------------------------------------------------------------------------------
　1. [参照] ボタンをクリックし、画像ファイルのあるフォルダーを選択します。

　2. 表示したい画像ファイルをチェックします。

　3. [詳細設定] ボタンをクリックし、自分の用途に適した設定をします。

　4. [作成] ボタンをクリックし、作成開始確認が出るので問題なければ [はい] ボタンをクリッ
    クします。

　5. 作成フォルダーを設定します。問題なければ [OK] ボタンをクリックします。

　6. 処理が完了すると、確認メッセージが表示されます。

　※本ソフトの動作は、すべてのファイル(フォルダー)を保証しているものではありません。
　※その他の詳細な操作方法は、ヘルプファイル（ThumbnailAlbum.chm）をご覧ください。
